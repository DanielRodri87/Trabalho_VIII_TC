
------------------------

- (adicione aqui outras pastas/arquivos relevantes)

Instalação rápida
-----------------
1. Clonar o repositório:
```bash
git clone <url-do-repositorio>
cd Last_Trabalho
```
2. Criar ambiente virtual (opcional mas recomendado):
```bash
python -m venv .venv
source .venv/bin/activate  # Linux / macOS
.venv\Scripts\activate     # Windows
```
3. Instalar dependências:
```bash
python -m pip install -r requeriments.txt
```

Uso
---
- Instruções gerais sobre como executar scripts ou notebooks.
- Exemplo de execução (substitua pelos scripts reais do projeto):
```bash
python src/main.py --input data/dataset.csv --output results/
```

Como reproduzir os resultados
-----------------------------
1. Garanta que os dados estejam na pasta `data/` (ou ajuste o caminho).
2. Execute os notebooks ou scripts na ordem indicada nos comentários de cada arquivo.
3. Salve os artefatos gerados em `results/` para comparação.
